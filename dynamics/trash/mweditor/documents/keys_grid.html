Keys in world grid mode

BFill the first floor of selected region with the selected code.
UFill the second floor of the selected region with the selected code.
Del.Clear both floors of the selected region.
RStart/Stop the rendering view.
Ctrl-CCopy the selected region into the clipboard.
Ctrl-VPaste the clipboard starting at the top-left point of the selected region.
Ctrl-ZUndo the last drawing action.
