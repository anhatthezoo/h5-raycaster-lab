Draw some wall tiles

Use an external painting software (like Gimp). Create an image of size 256 x 96 pixels. This image will contain wall tiles we need to paint our level. Each wall tile is 64 x 96 pixels. So you can draw up to 4 tiles in this 256 pixel wide image.

I drew 2 tiles : a brick wall and a wooden door. There is room for 2 more tiles.
