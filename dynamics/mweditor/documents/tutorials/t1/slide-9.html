Creating a wall block

The wall block will allow to define non-crossable walls.

  1. Create a new block (+ button), name it "wall1".
  2. In the Wall tiles window, select a Left tile Left mouse button.
  3. Select a Right tile with Right mouse button.
  4. In the Block builder window, select the physic code "Solid wall".
Walls have 4 sides and can have 2 different tiles. The Left tile will be painted on the Y-aligned sides, the Right one will be on the X-aligned ones.
