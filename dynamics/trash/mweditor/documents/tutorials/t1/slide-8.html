Creating a floor block

Let's create our first block. The floor block is used to define walkable areas.

  1. Create a new block (+ button), name it "floor1".
  2. In the Flat tiles window, select a Floor tile with Left mouse button.
  3. Select a Ceil tile with Right mouse button.
  4. In the Block builder window, select the physic code "Void".
Your floor block is ready to use on the grid.
