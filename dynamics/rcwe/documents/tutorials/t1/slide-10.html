Creating a door block

The door block is used to define doors, which can be open, and automaticaly close after a few seconds.

  1. Create a new block (+ button), name it "door1".
  2. In the Wall tiles window, select the same tile withLeft then Right mouse buttons. No need to have different tile for this type of block.
  3. In the Flat tiles window, select a floor tile and ceil tile. Because floor and ceil will be visible when the door opens.
  4. In the Block builder window, select the physic code "Door double".
Depending of your door tile design, you may choose different physic codes (Door up, Door left, Door right...). This option varies the open and close doors animations.
